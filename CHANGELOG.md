# Changelog

All notable changes to the Ultahost DNS Plugin will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Ultahost DNS Plugin
- PowerDNS v4 API integration
- WHM settings interface for root configuration
- DNS zone creation and deletion
- DNS record management (add, update, delete)
- Support for all DNS record types (A, AAAA, MX, CNAME, TXT, SRV, etc.)
- Permission-based access control (Root/Reseller/User)
- DNS template integration
- Comprehensive logging system
- Connection testing utility
- Installation and uninstallation scripts
- Test suite

### Features
- Complete replacement of cPanel/WHM default DNS module
- Automatic zone creation with DNS templates
- Secure configuration management
- Error logging and debugging support
- Compatible with cPanel/WHM 130.x.x
